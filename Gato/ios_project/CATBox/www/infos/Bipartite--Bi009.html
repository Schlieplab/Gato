This algorithm finds a maximum matching in a bipartite graph with partition V (red) and W (green). To proove the result it will show a minimum edge cover that has (in bipartite graphs) the same cardinality as the maximum matching (see Chapter 8 of the book).

Visualization

Vertex Colors
    
Initial color of the vertices. Color of one color class when the bipartition has been computed.
    
Color of the other color class when the bipartition has been computed.
    
Color of vertices that are in the queue Q or have been removed from the queue in the present procedure to search for an augmenting path.
    
Vertices of the red color class that have been marked in the present procedure to search for an augmenting path.
    
The vertex being processed is displayed with a wide outline in this color.
When a maximum matching has been found we use the following colors show the vertex cover.
    
Covering vertices from the red color class.
    
Covering vertices from the green color class.
    
All covering vertices are displayed with a wide outline in this color.

Edge Colors
    
Initial color for all edges in the very beginning.

    
Initial color for all edges when a search forest is grown.
    
Edges in a matching.
    
Edge that is currently processed.
    
Non-matching edges in the search forest.
    
Edges that have been processed but are not in the search forest.
Further elements
    
When an augmenting path is found we indicate this by a yellow shade around the edges of the path.
