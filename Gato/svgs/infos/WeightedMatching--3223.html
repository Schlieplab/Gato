This algorithm finds a maximum weight matching in a complete graph with Euclidean distances, given as set of points in the plane, by a primal-dual algorithm. One window visualizes the dual variables as moats, the primal variables as matching and the edges of the graph which have no slack. A secondary window (shrinking) depicts the graph, where the blossoms with non-zero dual variables are shrunken into a single vertex. On this graph a cardinality matching is performed as described in Chapter 8 of the book. It returns a maximum matching and a Gallai-Edmonds decomposition into a barrier, odd components and even components . This is used to update the dual variables, see Chapter 9 of the book.

Visualization

Primary Window

Vertex Colors
    
Unmatched vertices are dark blue.
    
Matched vertices are blue.
Edge Colors
    
Initial color for edges. These edges occur if the corresponding inequality of the dual program is tight, they do not belong to matching corresponding to the primal solution.
    
Color of edges belonging to the current matching.
    
Color of edges belonging to an old matching along an augmenting path.
Further elements
    
The size of a red circle indicates the value of the dual variables associated with the inner (blue) vertex.
    

Secondary Window

This window depicts the graph, where the blossoms with non-zero dual variables are shrunken into a single vertex. On this graph a cardinality matching is performed as described in Chapter 8 of the book. It returns a maximum matching and a Gallai-Edmonds decomposition into a barrier , odd components and even components .

Vertex Colors
    
Vertices belonging to the barrier are red.
    
Vertices in an odd component are dark blue.
    
Vertices in an even component are blue.

Edge Colors
    
Initial color for edges. These edges occur if the corresponding inequality of the dual program is tight, they do not belong to matching corresponding to the primal solution.
    
Color of edges belonging to the current matching.
