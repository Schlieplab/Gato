Minimal Spanning Tree - Prim

This algorithm finds a minimal spanning tree in a connected and undirected graph.

Visualization

Vertex Colors
    
Initial color for vertices.
    
Vertices for which the incident edge is in F and no incident edge is in T.
    
Vertices which have at least one incident edge in T.
    
A wide outline in this colors designates the vertex which neighborhood the algorithm currently explores.
Edge Colors
    
Initial color for edges.
    
Color of the edge being considered for inclusion in F (transient).
    
The minimum weight edge in F (transient).
    
An edge removed from F (transient).
    
Edges which are part of the MST.
    
Edges which are not part of the MST.
