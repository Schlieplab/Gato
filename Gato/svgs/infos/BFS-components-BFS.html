This algorithm finds connected components in a graph by performing one breadth-first-search per component. It visualises the predecessor tree being built and the BFS vertex labels. See Chapter 2 in the CATBox book.

Visualization

Vertex Colors
    
 
Initial color for vertices that have neither been visited nor processed.
    
 
Vertices which have been visited as a neighbor while processing the active vertex.
    
 
First component only: Vertices of this color have been processed; i.e., their neighbors have been explored. Processed vertices will get different colors in other components.
    
 
The vertex being processed is displayed with a wide outline in this color.
Edge Colors
    
 
Initial color for edges. Edges of this color have not been traversed yet.
    
 
Color of the edge being traversed.
    
 
First component only: Traversed edges displayed in the color are part of the BFS tree. A different color will be used in other components
    
 
Color of edges traversed which are not part of the DFS tree.
Further elements
    
 
The BFS label assigned to visited vertices is displayed in this color.
