This algorithm finds a maximum cardinality matching in a not necessarily bipartite graph. One window depicts the graph and the actual matching. In a secondary window (shrinking) we search for an augmenting path or a new blossom in a graph, where the blossoms have been shrunken into a single vertex, see Chapter 8 of the book.

Visualization

Primary Window

Vertex Colors
Unmatched vertices are grey.
Matched vertices are blue.
When a maximum matching has been found we use the following colors to show the Gallai-Edmonds decomposition.
Vertices in the barrier are red.
Vertices in an odd component are orange.
Vertices in an even component are blue.
Edge Colors
    
 
Initial color for edges.
    
 
Color of edges belonging to the current matching.
    
 
Color of edges belonging to an old matching along an augmenting path.
Further elements
    
 
When a blossom is found we indicate this by a yellow shade around the vertices that belong to the blossom.

Secondary Window

This window depicts the graph, where the blossoms are shrunken into a single vertex. Starting in unmatched vertices we grow an alternating search forest see Chapter 8 of the book.

Vertex Colors
Vertices not belonging to the search forest.
Other colors are used to indicate components in the search forest.

Edge Colors
    
 
Initial color for edges.
    
 
    
 
Color of a suspect edge connecting two outer vertices. This either creates a new blossom or connects two components of the search forest, thus establishing an augmenting path.
    
 
Color of edges belonging to old matching along an augmenting path.
Further elements
A label 1 at a vertex indicates an outer vertex in the search forest, a label 0 an inner vertex.
