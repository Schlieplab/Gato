Gato needs Python and Tcl/Tk. If you have never installed Gato before, you might want to check the installation page for information before proceeding. If you have installed Gato before, and just want to upgrade to a newer version, download the compressed archive file. Remember to download it as a binary.

CVS Access

If you want to check out the very latest version you can use anonymous CVS access. First login 

  > cvs -d:pserver:anonymous@gato.cvs.sourceforge.net:/cvsroot/gato login
using an empty password (just hit enter) and then check out Gato 
  > cvs -z3 -d:pserver:anonymous@gato.cvs.sourceforge.net:/cvsroot/gato co Gato
If you are registered developer you can download Gato using 
  > cvs -z3 -d:ext:username@gato.cvs.sourceforge.net:/cvsroot/gato co Gato
Dont forget to 
  > export CVS_RSH=ssh
